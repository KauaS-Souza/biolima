Lista de passos para crair um html

  1. Abra um editor de texto.
  2. Crie um novo arquivo.
  3. Escreva a estrutura básica.
  4. Salve o arquivo.
  5. Nomeie com extensão .html.
  6. Abra no navegador.
  7. Visualizar e editar.
  8. Use Identação (Recuo).
  9. Tags Semânticas.
  10. Atributo alt em Imagens.

Lista de um checklist para entregar uma atividade.

  1. Ler o tema.
  2. Enteder o tema.
  3. Pesquisa o tema.
  4. Tirar dulvida do tema.
  5. Ver os requisitos da atividade.
  6. Pegar os requisitos pedidos da atividade.
  7. Checar se a data e hora de entrega estão corretas (atenção ao fuso horário).
  8. Incluir o destinatário correto ou enviar para a plataforma correta.
  9. Organizar os arquivos de trabalho e fazer backup.
  10. Celebrar a conclusão e atualizar status em ferramentas de gestão.
